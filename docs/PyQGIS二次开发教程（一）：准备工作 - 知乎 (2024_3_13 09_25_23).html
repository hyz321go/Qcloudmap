一 前言

随着GIS应用在国内的逐渐增多，越来越多的行业会需要使用GIS软件进行处理。目前国内的GIS软件基本为ArcGIS，ESRI作为GIS行业的龙头老大，依靠着ArcGIS基本占据着GIS软件的全部江山，其功能丰富强大，且社区最为活跃，但随着近期的俄乌战争，ESRI制裁俄罗斯的事件，还是可以感受到闭源软件的不可控性。

QGIS是一个开源、免费、轻量级的GIS软件，虽然在国内的普及程度不高，但不妨碍其软件的优秀程度。QGIS的软件功能及其丰富，并且还有海量的插件库供不同方向的GIS人员选择，是一个充满活力的开源社区。QGIS是一个使用C++语言开发的软件，其二次开发难度较大，且需要经历复杂的编译、源码理解，需要较高的门槛。

值得庆幸的是，QGIS支持python语言进行二次开发， python作为一个较易上手的胶水语言，在GIS社区较为活跃，例如：GDAL、Shapely、Fiona、RTree、pyproj、GeoPandas、rasterio、GeoDjango等开源框架都可以在python上使用，python语言是一个特别适合GIS领域的语言，更不用说现在火热的机器学习、深度学习都与python密切相关。QGIS的提供的PYQGIS的接口与C++ QGIS的接口基本保持一致，学习PyQGIS也能基本熟悉c++的QGIS，方便后续继续进行深入研究，作为想要入门QGIS二次开发的人员来说再为合适不过了。

本系列的教程将会从二次开发环境配置、pycharm配置、pyqgis语法、qtDesigner使用教程、插件设计等方面，结合笔者在项目中的实践一一展开叙述，希望最后，能在所有的系列教程结束后，让跟着系列做的读者都可以开发出一个属于自己的GIS界面。

学习本教程的前提为如下：

  • 基本了解python语法，若不了解，请自行查阅python入门方法
  • 基本了解qgis的使用方法，若不了解，请按照后续说明安装qgis后进行学习
  • 大致了解pyqt的基本使用，若不了解，也可以在PyQGIS的学习中慢慢了解

本节教程为PyQGIS的二次开发准备工作，在本节教程结束后，读者应该完成了qgis安装、 Pycharm环境配置、pyqt工具配置、Qt Designer工具配置，并能够简单运行出一个简单的GIS界面。

二 QGIS安装与 Pycharm安装

2.1 QGIS安装

在Index of /downloads (qgis.org)这个网址可以找到QGIS的下载地址，笔者使用了稳定版本QGIS 3.22LTR，点击msi进行安装，可以打开qgis的文件夹查看是否为这个文件结构：

记住这个bin文件夹，我们在后面的pycharm环境配置时需要使用该文件夹。我们在bin中点击qgis-ltr-bin.exe，打开qgis应用程序，输入Ctrl + Alt + P，打开Python控制台，我们尝试输入一下代码：

from qgis.core import QgsRasterLayer

若没有提示报错，说明我们的安装成功了。

2.2 Pycharm安装

Pycharm是一种PythonIDE，带有一整套可以帮助用户在使用Python语言开发时提高其效率的工具，比如调试、语法高亮、Project管理、代码跳转、智能提示、自动完成、单元测试、版本控制。在PyCharm: the Python IDE for Professional Developers by JetBrains官方网站中可以下载Pycharm的Community版本，该版本免费使用（当然如果你想支持Pycharm，购买专业版也可以~）

三 Hello QGIS

我们现在来尝试一下在独立脚本中使用PyQGIS，但在启动前，我们还需要查看一下QGIS的系统安装路径，如果之前读者是一步一步跟着教程过来的话，应该会很清楚QGIS的安装路径，如果不清楚的话，也可以在QGIS自带的控制台中找到，输入Ctrl + Alt + P，打开Python控制台，并输入QgsApplication.prefixPath()：

我们在项目的main.py中输入如下代码：

from qgis.core import QgsApplication
​
# 告知QGIS路径
QgsApplication.setPrefixPath('C:/qgis322/apps/qgis-ltr', True)
​
# 第二个参数为是否启用GUI
qgs = QgsApplication([], False)
​
# 初始化QGIS
qgs.initQgis()
​
# 开始写代码
print(QgsApplication.prefixPath())
print("Hello Qgis！")
​
# 从内存中删除数据提供程序和层注册表来结束
qgs.exitQgis()

4.2 界面设计

下面我们来进行界面设计。我们首先需要调研一个gis软件的常见界面设计，我们这里以ArcMap 10.7为例：

我们可以分析出，ArcMap的主界面主要分为：

  1. 图层树
  2. 地图画布
  3. 菜单栏
  4. 工具栏
  5. 状态栏

其中最重要的部件为图层树和地图画布，软件使用者首先会用不同方式将遥感影像或矢量拖入图层树中，此时，地图画布将会根据图层树的前后顺序渲染遥感影像和矢量并显示在地图画布中。菜单栏将会存放软件的所有功能，而工具栏将会放置一些常用的功能；状态栏则会实时展示一些如鼠标所在位置的XY坐标（经纬度）等实时状态。本次教程的参考就是gis软件的龙头老大：ArcMap，我们将以他的界面逻辑来进行二次开发讲解。

五 Pycharm External Tools 外部工具加载

我们上一节说明了最基本的窗体制作，但是现在我们还不能进行调用，我们需要将.ui文件和.qrc资源文件都转换为.py文件。pyqt提供了pyuic和pyrcc工具来进行转换，而Pycharm提供了一个外部工具加载方法，我们打开File | Settings | Tools | External Tools ：

5.1 增加pyuic工具

我们在打开File | Settings | Tools | External Tools后，点击左上角的+号，进行配置填写：

  • Name输入pyuic，
  • Programs输入 你自己的qgis应用地址/bin/python-qgis-ltr.bat，
  • Argument输入 -m PyQt5.uic.pyuic $FileName$ -o $FileNameWithoutExtension$.py ,
  • Working directory输入 $FileDir$

然后通过右键myWindow.ui，选择exterl-tools，pyuic：

运行后，myWindow.ui同级目录下会生成myWindow.py文件

5.2 增加pyrcc工具

我们在打开File | Settings | Tools | External Tools后，点击左上角的+号，进行配置填写：

  • Name输入pyrcc，
  • Programs输入 你自己的qgis应用地址/bin/python-qgis-ltr.bat，
  • Argument输入-m PyQt5.pyrcc_main $FileName$ -o $FileNameWithoutExtension$_rc.py ,
  • Working directory输入 $FileDir$

然后通过右键myRc.qrc，选择exterl-tools，pyrcc,我们会在resources文件夹生成myRc_rc.py,我们将其移到根目录下，并且右键项目，将其设为Source Root

六 运行属于自己的GIS软件界面

6.1 使用继承解耦代码

我们现在确实创建了myWindow.py文件，但软件开发不可避免的是，我们会常常改动ui文件，所以我们不能在创建出来的myWindow.py上进行代码编写，我们应该继承这个代码里的对象，在继承对象中编写代码，我们在根目录下新建mainWindow.py：

from qgis.PyQt.QtWidgets import QMainWindow
from qgis.core import QgsProject
from ui.myWindow import Ui_MainWindow
​
PROJECT = QgsProject.instance()
class MainWindow(QMainWindow, Ui_MainWindow):
    def __init__(self):
        super(
MainWindow, self).__init__()
        self.setupUi(self)

我们在这个新子类中，继承了多个父类，一个是QMainWindow，这个父类表明了这个类的实现是一个主窗口对象；第二个是Ui_MainWindow,从导包情况可以看到，这个类出自于ui.myWindow，就是我们自己转换来的myWindow.py的类。

6.2 尝试运行自己的界面

我们继承好类后，需要有人调用，并创建为对象，我们在根目录的main.py将代码改写为如下：

from qgis.PyQt import QtCore
from qgis.core import QgsApplication
from PyQt5.QtCore import Qt
import os
import traceback
from mainWindow import MainWindow
​
if __name__ == '__main__':
    QgsApplication.setPrefixPath('C:/qgis322/apps/qgis-ltr', True)
    QgsApplication.setAttribute(Qt.AA_EnableHighDpiScaling)
    app = QgsApplication([], True)
    app.initQgis()
    mainWindow = MainWindow()
    mainWindow.show()
    app.exec_()
    app.exitQgis()

运行main.py，如果弹出下面的窗口，恭喜你，已经可以运行自己的界面啦！

虽然将界面运行出来了，但现在该窗口只是一个空壳，软件使用者不能在该界面中做任何操作，下一个文章，我们将学习如何绑定图层树、如何绑定地图画布、如何加载栅格文件和矢量文件。
